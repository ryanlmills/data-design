Interaction Flow

  • Duncan enters letterboxd.com into his browser.
  • Server returns the main Letterboxd page with Duncan's account logged in.
  • Duncan clicks the "+ LOG" icon.
  • Server displays Search prompt.
  • Duncan enters "The Red Tent".
  • Server suggests the film in a list.
  • Duncan clicks the correct option.
  • Server displays new prompt to review film.
  • Duncan fills out form checkboxes, date, rating, and review field, clicks "SAVE".
  • Server closes review prompt, completes request to save review, displays completion notification in the upper left-hand corner.
  • Duncan mouses over username.
  • Server displays drop-down menu of options.
  • Duncan clicks "Films".
  • Server loads Duncan's Films page on profile.
  • Duncan clicks "REVIEWS".
  • Server loads Duncan's Reviews page on profile.
  • Duncan sees that the review is viewable.

Frequency of Use

: Daily
